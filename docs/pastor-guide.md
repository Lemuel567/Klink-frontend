
# Pastor Guide

This guide explains how pastors use the Klink app.

## What You Can Do

- **Announcements** — post church-wide announcements that all members see
- **Sermons** — upload and manage the sermon archive
- **Attendance** — view attendance records for services and events
- **Prayer Wall** — view prayer requests submitted by members
- **Members** — view the member directory
- **Groups** — oversee small groups and ministries
- **Projects** — create and track church projects and fundraising goals

## Getting Started

1. Log in with your pastor account
2. Use the home screen to navigate between sections
3. Post an announcement from the Announcements tab whenever there's news
   the congregation should know
4. Upload new sermons after each service so members can catch up later
