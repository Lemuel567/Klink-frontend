# Klink Frontend

React Native mobile app for the Klink church management platform. Built with Expo, featuring stunning animations, parallax effects, glassmorphism, and a premium church worship visual experience.

## Tech Stack
- React Native
- Expo SDK 52
- TypeScript
- Expo Router
- React Query
- Zustand
- React Native Reanimated
- Axios

## Features
- Beautiful worship-themed UI
- Parallax scroll effects
- 3D animations
- Glassmorphism cards
- Dark mode support
- Push notifications
- SMS verification
- Complete church management features

## How to Run
```
npm install
npx expo start
```

## Testing on Device
- Android emulator: `http://10.0.2.2:8080/api/v1`
- iPhone on hotspot: `http://172.20.10.3:8080/api/v1`
