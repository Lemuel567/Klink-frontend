# Team Task Guide

Before your first commit, set YOUR identity so the work counts on YOUR profile:

```
git config --global user.name  "Your Name"
git config --global user.email "the-email-on-YOUR-github-account"
```

Workflow: branch (`<name>/<task>`) -> small commits (one per document/test/fix)
-> push -> Pull Request -> lead reviews & merges. Spread work over days.

Assignments live in the GitHub Issues of the two repos:
- Klink-frontend: QA test pass · README & user guides
- Klink-backend: unit tests · deployment/privacy docs & presentation
